===============================================================================
       AKRITI DIAGNOSTICS CENTER — STEP-BY-STEP LIVE DEPLOYMENT GUIDE
===============================================================================

This guide explains how to deploy your FastAPI Backend + HTML Frontend + PostgreSQL
database live on the web for FREE using Render and Supabase.

-------------------------------------------------------------------------------
PREREQUISITES:
-------------------------------------------------------------------------------
1. A GitHub Account (https://github.com)
2. Git installed on your computer.
3. The project code pushed to a repository on your GitHub account.

-------------------------------------------------------------------------------
STEP 1: DATABASE SETUP (SUPABASE)
-------------------------------------------------------------------------------
Supabase provides a persistent, free PostgreSQL database. It does not expire.

1. Go to https://supabase.com and log in using your GitHub account.
2. Click "New Project" and choose your organization.
3. Fill in the project details:
   - **Name**: Akriti Diagnostics
   - **Database Password**: Set a secure password. Save it somewhere safe!
   - **Region**: Choose a region closest to your users (e.g., "South Asia (Mumbai)").
   - **Pricing Plan**: Select "Free".
4. Click "Create new project". Wait 2-3 minutes for the database to provision.
5. Once ready, click the green "Connect" button at the top of the Supabase dashboard.
6. Click the "Direct" tab and copy the connection string. It looks like:
   postgresql://postgres.[YOUR_PROJECT_ID]:[YOUR_PASSWORD]@aws-0-[REGION].pooler.supabase.com:5432/postgres
7. Replace `[YOUR_PASSWORD]` in that string with the database password you saved.
   *IMPORTANT*: URL-encode any special characters in your password (e.g. `@` becomes `%40`).
   Your final URL should look like:
   postgresql://postgres:Kunal%4096086@db.dzxkjbfjrjtpqnerbrqa.supabase.co:5432/postgres

-------------------------------------------------------------------------------
STEP 2: BACKEND AND FRONTEND SETUP (RENDER)
-------------------------------------------------------------------------------
Render runs your FastAPI application (which hosts the backend and serves the
frontend HTML pages) on a free persistent Web Service.

1. Go to https://render.com and sign up/log in using your GitHub account.
2. On the dashboard, click the blue **New +** button (top right) and select **Web Service**.
3. Select **Build and deploy from a Git repository** and click Next.
4. Locate your repository (e.g. `Kunal9608/Akriti`) and click **Connect**.
5. Set the service configuration details:
   - **Name**: `akriti-diagnostics-backendd` (or any custom name)
   - **Environment / Runtime**: `Python`
   - **Branch**: `main`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python main.py`
   - **Instance Type**: Select the **Free** tier.

-------------------------------------------------------------------------------
STEP 3: CONFIGURING ENVIRONMENT VARIABLES (IN RENDER)
-------------------------------------------------------------------------------
Before deploying, click the **Advanced** button in the setup form, then add these 
Environment Variables:

1. **`DATABASE_URL`**: 
   `postgresql://postgres:Kunal%4096086@db.dzxkjbfjrjtpqnerbrqa.supabase.co:5432/postgres`
2. **`JWT_SECRET_KEY`**: 
   `KunalAkritiSecret2026` (or any long random secure string)
3. **`ENVIRONMENT`**: 
   `production`
4. **`PORT`**: 
   `8000`
5. **`HOST`**: 
   `0.0.0.0`
6. **`ALLOWED_ORIGINS`**: 
   `http://localhost:8000,http://127.0.0.1:8000,https://akriti-diagnostics-backendd.onrender.com`

-------------------------------------------------------------------------------
STEP 4: DEPLOY AND MONITOR LOGS
-------------------------------------------------------------------------------
1. Click **Create Web Service** at the bottom of the form.
2. Render will automatically check out your code, read the `.python-version` file, 
   load Python 3.11.9, install all packages from `requirements.txt`, and start the app.
3. The server will automatically connect to Supabase, run database setup tables, 
   seed initial admin/staff credentials, and start listening on port 8000.
4. Once Render marks the service as **Live** (green status), visit:
   `https://akriti-diagnostics-backendd.onrender.com` to see your running system!
